## windows环境下使用virtualenv

1、创建一个文件夹
2、启动命令行并进入该文件夹，键入如下命令（前提是你已经安装了python）：

```cmd
pip install virtualenv     # 安装 virtualenv 
```


3、在想要创建的文件夹内，键入如下命令：

```cmd
virtualenv venv            # venv 为创建的虚拟环境名，可自定义
```
4、激活虚拟的环境，在创建虚拟环境的文件夹内，键入如下命令：

```cmd
venv\Scripts\activate      # 还是老规矩，venv是虚拟环境名
```
