### 开启归档日志

```sql
--关闭数据库
shutdown immediate; 
--以mount状态打开数据库
startup mount;
--开启归档日志
alter database archivelog;
--开启数据库
alter database open;
--查看归档日志是否开启
archive log list; 
```

　　

### 关闭归档日志

```sql
--关闭数据库
shutdown immediate;
--打开数据库
startup mount;
--关闭归档日志
alter database noarchivelog;
--开启数据库
alter database open;
--查看归档日志是否关闭
archive log list;
```
